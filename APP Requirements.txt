What it does:

Reads Kite order CSV from a file path, pasted CSV text, or uploaded CSV.
Displays parsed/normalized orders in a table.
Lets you select all or specific trades to execute.
Supports dry run mode.
Lets you set these for the running app process:
KITE_API_KEY
KITE_API_SECRET
KITE_ACCESS_TOKEN
KITE_CONFIRM_LIVE_ORDER
Uses your existing src/script/kite_place_order.py logic for CSV parsing, price/quantity resolution, placing, and modifying similar open orders.
Does not hardcode the secrets into source.
